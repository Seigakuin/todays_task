### pseudo code (疑似コード)
1. 「あなたはどの色が好きですか？(黒 / 白 / 赤)」とユーザーに聞く
2. ユーザーが色を入力する
3. 現在のアンケート結果を出力する
4. 「調査をやめますか？(はい/いいえ)」をユーザーに聞く

### エラー対策
1. ユーザーが黒白赤以外の色を入力した場合
    - 「その色は調査していません。」と出力し、ループを継続
2. ユーザーが (はい/いいえ) 以外を入力した場合
    - ループを続ける
