p5.jsとは？

Made with Javascript

p5jsはJavascriptを使用したライブラリです。Javascriptはウェブ上のサイトを動かしている言語です。現在でウェブだけでなく、デスクトップアプリ、モバイルアプリなどの分野でも活躍しています。

VISUAL LIBRARY

p5jsは主にVisualに特化したライブラリで、Javascriptを使い、デジタルアートの描画や、ゲームなどを作ることができます。本来、映像プログラミングに必要な複雑なプログラミングをしなくて良いように設計されているライブラリです。
