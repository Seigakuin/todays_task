聖学院プログラミング部

活動内容

聖学院のプログラミング部では、毎週木曜日に活動をしネット上にあるサイト（p5.js）を利用し部活動のたびに先生から課される課題をこなすことで、JavaScriptなどのプログラミング能力の向上を目指しています。

プログラミング体験

今回のプログラミング体験では、
①絵の色を変える体
②動き回るボールの速さや動き、大きさを変える体験
③ボールを飛ばして行うブロック崩しゲーム
の計３つをご用意しています。それぞれのプログラムを自分の思う通りに改造してみましょう。
1年間活動をして、プログラムを作ってきました。ぜひ楽しんでいってください。

    • 水野幸

  • 説明

    • このページは家の色を変えることが出来ます。
    このゲームの難易度は易しいのであまり操作に慣れていない方でも簡単にできるので気軽に遊んでみてください。

    操作方法
    step1　指定の場所の数字を半角で0から255の中の好きな数字に書き換えてください。
    　　　 ↓↓下のURLはカラーコード表です
    　　　 https://itsakura.com/html-color-codes
    　　　 （255以上を入れると255としてカウントされます。）
    step2　書き換えたら左上の”▶”ボタンを押してください。
    step3　また数字を変えて、左上”▶”を押して色を変えて遊んでみてください。

p5.jsとは？

Made with Javascript

p5jsはJavascriptを使用したライブラリです。Javascriptはウェブ上のサイトを動かしている言語です。現在でウェブだけでなく、デスクトップアプリ、モバイルアプリなどの分野でも活躍しています。

VISUAL LIBRARY

p5jsは主にVisualに特化したライブラリで、Javascriptを使い、デジタルアートの描画や、ゲームなどを作ることができます。本来、映像プログラミングに必要な複雑なプログラミングをしなくて良いように設計されているライブラリです。
